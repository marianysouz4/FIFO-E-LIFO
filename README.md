# Fifo-E-Lifo
Estudo prático sobre FIFO e LIFO - conceitos básicos de pilhas e filas em estruturas de dados e suas diferenças.

#### O que são FIFO e LIFO?
O LIFO e o FIFO são dois tipos de estruturas de dados bastante usados na programação.

##### Qual o conceito de FIFO?
FIFO (First-In, First-Out) significa  "primeiro a entrar, primeiro a sair", e é definido como uma estrutura de fila, já que o primeiro elemento adicionado na fila é processado primeiro 

##### Qual o conceito de LIFO?
LIFO (Last-In, First-Out) significa “último a entrar, primeiro a sair", onde é definido como uma pilha, já que o elemento adicionado recentemente é o primeiro elemento a ser processado.

A principal diferença entre as duas estruturas de dados é a ordem de processamento.

##### Onde podemos utilzar o FIFO?
1. Supermercados
2. Farmácias

###### Breve explicação:
Vamos supor que existe uma empresa de alimentos onde a empresa recebe um lote de mercadoria no início do mês e, depois de 10 dias, recebe outro lote. Ele deve priorizar o primeiro lote, que está há mais tempo no estoque. Então nesse caso ocorre a estrutura de fila, já que o primeiro lote de alimentos deve ser o primeiro lote a sair, independente do recebimento de outro lote.

##### Onde podemos utilizar o LIFO?
1. Lojas de roupas
2. Lojas de carros

###### Breve explicação:
Vamos supor que uma loja de roupa acabou de lançar uma nova coleção de verão. As peças de roupa da nova coleção tem tendência em sair primeiro, pois são peças novas. Isso resume a estrutura de pilha, já que a última peça de roupa adicionada recentemente é a primeira a ser vendida.
